# CoreAIpet 产品需求文档（PRD）

## 产品名称

CoreAIpet

## 产品定位

面向开发者、项目经理和知识工作者的桌面AI助手。

---

# V1 - Desktop Pet

目标：

打造稳定可用的桌宠框架。

核心功能：

* Live2D角色
* 拖拽
* 系统托盘
* 悬浮菜单
* 模型切换
* 状态机
* 窗口位置记忆

成功指标：

* CPU < 5%
* Memory < 150MB

---

# V2 - AI Assistant

目标：

让桌宠具备对话能力。

新增：

* Chat面板
* OpenAI兼容模型
* 本地会话存储
* 长短期记忆
* Agent状态管理

核心场景：

* 问答
* 工作建议
* 总结文档

---

# V3 - Work Assistant

目标：

成为工作入口。

新增插件：

* Jira
* Slack
* Gmail
* Outlook
* Calendar
* DingTalk
* Feishu

能力：

* 自动晨报
* 自动日报
* Sprint总结
* 邮件摘要

核心用户：

* PM
* Tech Lead
* Developer

---

# V4 - AI Workforce

目标：

多Agent协作平台。

新增：

* Jira Agent
* Email Agent
* Meeting Agent
* Research Agent

表现形式：

* Live2D模式
* Pixel Office模式

场景：

* AI员工协同办公
* 自动项目跟踪
* 自动风险分析
* 自动会议安排

最终形态：

AI Agent Startup Dashboard
+
Desktop Assistant
+
Work Memory Platform
